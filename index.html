Tecnologías dominadas

HTML,CSS y JS

Bases del desarrollo web. HTML para estructura, CSS para diseño y JS para interactividad. Juntos crean sitios modernos y atractivos. Esenciales para desarrolladores web.

Laravel

Laravel es un potente framework de código abierto en PHP, con una sintaxis elegante y herramientas que simplifican tareas comunes como manejo de rutas, bases de datos y autenticación, entre otras.

MySQL

Motor de base de datos relacional de código abierto ampliamente utilizado en aplicaciones web y empresariales de mediana y gran escala.

SqlServer

Motor de base de datos relacional desarrollado por Microsoft. Es ampliamente utilizado en empresas para el manejo de grandes volúmenes de datos y el soporte de aplicaciones empresariales.

Node Js

Entorno de tiempo de ejecución de JavaScript fuera del navegador. Ideal para crear servidores web y aplicaciones en tiempo real. Enfoque no bloqueante y escalabilidad. Amplio ecosistema de módulos (npm).

Livewire

Framework de desarrollo web para Laravel. Crea interfaces interactivas con solo PHP, sin JavaScript. Facilita la construcción de aplicaciones modernas y atractivas.

Vue JS

Vue.js es una biblioteca de JavaScript para crear interfaces de usuario. Es un framework ligero y fácil de aprender, y se puede usar para crear aplicaciones web modernas.
